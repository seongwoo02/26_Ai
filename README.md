# Arrow Dodge

맵 바깥에서 날아오는 화살을 피하며 오래 버티는 웹게임입니다.

## 실행

PowerShell에서 아래 순서로 실행합니다.

```powershell
cd E:\26_1Ai
python server.py
```

브라우저에서 `http://localhost:8000` 으로 접속합니다.

## 외부 공개

다른 터미널에서 ngrok을 실행합니다.

```powershell
ngrok http 8000
```

생성된 `https://...ngrok-free.app` 주소로 다른 사람도 접속할 수 있습니다.

## 기능

- 방향키로 플레이어 이동
- 시간이 지날수록 화살 수 증가
- 시간이 지날수록 화살 속도 증가
- 점수는 생존 시간이며 `10.2s` 형식으로 표시
- 게임 오버 후 닉네임 입력 및 리더보드 저장 가능
- 리더보드는 서버 파일에 저장되고 접속자 모두에게 갱신됨
